Gaussian Kernel (高斯核)

我们现在由于计算复杂度和维度没关系了，那么我们如果做feature tranform到一个无穷多维度的空间上会怎么样呢？

我们上图把$exp(-(x-x’)^2)$看做核，然后反着取做转化，化为$\phi(x)^T\phi(x’)$的形式，那么此时的$\phi(x)$竟然代表了一个无穷多维的转化。
